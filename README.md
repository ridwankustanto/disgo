# DISGO, Discourse HTTP API client in Go

This library is a [Discourse HTTP API](https://docs.discourse.org/) client for the Go language.

## Installation

```
go get github.com/ridwankustanto/disgo
```
Import the package:

``` go
import (
       "github.com/ridwankustanto/disgo"
)
```
